CSS Basics

CSS stands for Cascading Style Sheet.It is the designer of website.It adds colours to font,gives pretty layout.

This text is changed using CSS.

CSS file contains special tags called ' style 'inside the 'head' tag.
The general form is'style type="text/css".To change any element in the website in between the style tags we have to use the element name(name written inside "< >" bracket).
Then within the curly bracket"{ }" specify the property name and value followed by a semicolon.The whole form is
style type="text/css"
element name {
property name : value ;}style

Padding

Inside the border.General form :
background-color: colorName
Here the background is the background of given element.Another properties are padding-left: px, padding-right: px

Border

General form : 3 properties are needed
border-color: colorName
border-width: px
border-style: colorName.
Or everything can be specified in one line.
border:px,style,color

Margin

It surrounds entire element outside the border.General form : 3 properties are needed
border-color: colorName
border-width: px
border-style: colorName

Link

Link is composed of states which is basically when and how the link is pressed.
There is one state for when the page is first visited.There is another state for when th link is clicked.There is another state for as we mousing over the link.And the last state is as we clicking the link.General form :

This is a link.
a:link{ color: red; text-decoration:none; }
a:visited{ color:green; }
a:hover{ background-color: blue; color:white; text-decoration: underline; font: bold; }
a:active{ background-color: orange;

Line-height

This property makes the website more readable by creating spaces among lines.
eg. line-height: 150%

Add an image in background

The element name will be the 'body'.General form :
background-image:url(fileName.extension)
To remove repeat: background-repeat: no-repeat
To remove repeat at certain direction:repeat-x or repeat-y
To place the image at a certain position: background-position: 95% 5px;
